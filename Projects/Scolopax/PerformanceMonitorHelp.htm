How to use it:

Query Input tab

  1. Select species of interest
    1. Use control+click to add another species
    2. Use shift+click to add range of species
  2. Select region and partitioning of interest
  3. Select a ranking measure for sorting the summaries
    1. Max-Min: Summaries with the greatest difference between min and max observation probability are first
    2. Standard Deviation: Summaries showing greatest variability in observation probability are first
    3. Slope of Regression Line: Summaries showing the strongest increasing trend are first
  4. Submit

Results tab

  1. The main graph shows the progress of our new approach compared to the previous state of the art.
    1. Since progress is measured through reports encoded in Hadoop global counters, true progress might be a bit ahead of the reported numbers.
    2. Stragglers affect completion time of the job in the end.
  2. Browse through the results to verify that both jobs computed the same (modulo minor differences due to numeric precision limits).
  3. The settings panel allows plot settings to be changed. To export a plot, un-select "show interactive plots".

At any time, it is possible to go to another tab by simply clicking on it. However, if the browser's forward or back buttons are used, the previous state is lost.
