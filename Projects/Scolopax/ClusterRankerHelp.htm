How to use it:

Query Input tab

  1. Select species of interest
    1. Use control+click to add another species
    2. Use shift+click to add range of species
  2. Select summary attributes of interest, i.e., the attribute on the x-axis of the summary. A set of one-dimensional summaries will be generated for each selected attribute
  3. Select region and partitioning of interest
  4. Choose a clustering algorithm
  5. Choose a distance function (Euclidean distance is most common.)
  6. Choose desired number of clusters (Recommendation: to see something interesting, choose a small number like 4.)
  7. Submit

Results tab

  1. Browse through the clusterings found using Previous and Next.
  2. Hovering over a colored box on the map shows the corresponding summary plot.
  3. Clicking on a colored box opens a pop-up window (make sure pop-ups are enabled in your browser) with the corresponding summary plot. Plots are placed on top of each other. Drag them around as desired.

At any time, it is possible to go to another tab by simply clicking on it. However, if the browser's forward or back buttons are used, the previous state is lost.
